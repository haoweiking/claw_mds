# 交易知识体系

## 一、核心策略

### IPA-Claw 策略
- **核心理念**: 预测"市场是否犯错"，而非预测价格
- **核心指标**: 理论胜率 vs 市场赔率
- **入场条件**: 偏差 > 7%
- **数据源**: Binance (CEX) + Polymarket

### V-MAC 策略
- **核心理念**: 趋势 + 动量 + 盘口
- **核心指标**: VWAP, EMA(9/21), RSI, 订单簿失衡
- **入场条件**: 多指标共振

## 二、技术指标

### 1. RSI (相对强弱指数)
- **计算**: 一定周期内上涨/下跌幅度比
- **信号**:
  - RSI > 70: 超买，可能反转
  - RSI < 30: 超卖，可能反弹
  - RSI = 50: 中性

### 2. VWAP (成交量加权平均价)
- **计算**: Σ(价格×成交量) / Σ成交量
- **用途**: 机构成本线
- **信号**:
  - 价格 > VWAP: 偏多
  - 价格 < VWAP: 偏空

### 3. EMA (指数移动平均)
- **用途**: 趋势跟踪
- **信号**:
  - EMA9 > EMA21: 金叉，多头
  - EMA9 < EMA21: 死叉，空头

### 4. 订单簿失衡 (Order Book Imbalance)
- **计算**: (买单总量 - 卖单总量) / (买单 + 卖单)
- **信号**:
  - 失衡 > 20%: 买单多于卖单，多头信号
  - 失衡 < -20%: 卖单多于买单，空头信号

## 三、风险管理

### 1. 仓位管理
- 单笔不超过账户 2-5%
- 建议 5-10% 仓位

### 2. 止损纪律
- 亏损 20% 必须止损
- 不扛单

### 3. 空仓过滤器
- 波动率极低时停止交易
- 没有波动 = 纯赌博

---

## 四、套利机会

### 1. 资金费率套利
- **原理**: 当资金费率极高(>0.03%)，市场过度看多
- **操作**: 反手做空

### 2. 预测市场套利
- **原理**: Polymarket 价格 vs 理论概率偏差
- **操作**: 偏差 > 7% 时入场

### 3. 延迟套利
- **原理**: 结算前30秒价格剧烈波动
- **操作**: 顺势追入

## 六、交易纪律

1. **不预测，只跟随**
2. **有信号才入场，无信号等待**
3. **亏损后不急于翻本**
4. **连续亏损3次，暂停交易**
5. **记录每一笔交易**
